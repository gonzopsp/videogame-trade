# *Flutter + Laravel + MySQL Full Stack Mobile App*
## videogame trade
### Descripción

- Videogame Trade es una aplicación móvil que permite a los usuarios comprar, vender e intercambiar videojuegos de forma sencilla y segura. La aplicación está desarrollada con Flutter para ofrecer una experiencia multiplataforma, mientras que el backend está construido con Laravel y utiliza una base de datos MySQL para la gestión robusta de datos.

Los usuarios pueden mantener una biblioteca virtual de videojuegos que poseen y ofrecer videojuegos para intercambio, retirándolos de su colección para facilitar el intercambio con otros usuarios.

### Tecnologías

  **Flutter** — Desarrollo móvil Android.

  **Laravel (PHP)** — API backend RESTful.

  **MySQL** — Base de datos relacional para almacenar usuarios, videojuegos, transacciones e intercambios.
